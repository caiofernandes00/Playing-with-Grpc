# Playing-with-Grpc

This is a simple project to play with Grpc features and golang.

Feature covered:

- Unary
- Server Streaming
- Client Streaming
- BiDi Streaming
- Interceptors w/ unary and streaming (authentication)
- Error handling (deadlines, status codes, etc)
- TLS
